# 📊 PhonePe Analysis Dashboard

## 📌 Project Overview
This project is a Power BI dashboard built using PhonePe transaction data to analyze transaction performance, user behavior, payment success rate, and service-wise contribution.

The dashboard provides key business insights through interactive visualizations and KPI cards.

## 🚀 Key Features

- Total Transactions Analysis
- Total Transaction Value Analysis
- Unique Users Tracking
- Success Rate Monitoring
- Month-wise Transaction Trends
- Age Segment Contribution Analysis
- Service-wise Transaction Value Analysis
- Top 5 Users by Transaction Value
- Weekday vs Weekend Usage Analysis
- Interactive Filters and Slicers

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## 📈 KPIs Included

- Total Transactions: 300K
- Total Value: ₹3.47 Billion
- Unique Users: 101K+
- Success Rate: 96%

---

## 📊 Dashboard Insights

- Weekday transactions contribute the highest share of overall activity.
- Millennials account for the largest user segment.
- Successful transactions exceed 95%, indicating strong platform reliability.
- Loans generate the highest transaction value among services.
